# Opening
o // show in current
go // show in current no switch
i, gi // split horizontally
s, gs  // split vertically
e // explore directory


# Directories
r // refresh cursor dir
R // refresh recursively
C // set root to selected item
cd  // change :pwd to selected dir
CD // change dir to :pwd
m // show menu for editing (create, delete, move, copy)
I // show hidden files
e // explore selected directory

# Bookmarks
B // show bookmarks
D // delete bookmark
:Bookmark // bookmark selected item
$HOME/.NERDTreeBookmarks  // bookmarks file
